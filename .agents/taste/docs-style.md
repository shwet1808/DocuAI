# Docs Style Taste

- Keep AGENTS.md concise (~75 lines) with directive keywords near the top; detailed rules belong in formal docs routed from AGENTS.md. Confidence: 0.90
- Use standard file names (SKILL.md, AGENTS.md, etc.) — do not re-invent naming conventions. Confidence: 0.85
- Prefer 2-3 focused docs over one monolithic doc when a topic crosses 300 lines. Confidence: 0.80
- README indexes must use stable links, heading anchors, and IDs, not manual line-count columns. Confidence: 0.90
